Do's and Don'ts

✓ Do's

  • Use lowercase "beacon"
  • Use Space Grotesk Bold for the 'b'
  • Use amber dot in wordmarks
  • Maintain clear space around logos
  • Use Inter for body text

✗ Don'ts

  • Don't use "Beacon AI"
  • Don't capitalize BEACON
  • Don't use other typefaces for the 'b'
  • Don't alter proportions
  • Don't use off-brand fonts
